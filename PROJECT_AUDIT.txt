================================================================================
                          ISSUES FOUND
================================================================================

------------------------------------------------------------------------------
CRITICAL ISSUES (Security/Functionality)
------------------------------------------------------------------------------

1. [SECURITY] Hardcoded Sentry DSN
   Location: src/main.jsx:8
   Issue: Sentry DSN is hardcoded in source code
   Risk: Exposes project identifiers publicly
   Fix: Move to environment variable (VITE_SENTRY_DSN)

2. [SECURITY] .env.sentry-build-plugin committed to repo
   Location: .env.sentry-build-plugin
   Issue: File is in .gitignore but exists in repository
   Risk: Sensitive build credentials may be exposed
   Fix: Remove from tracking with `git rm --cached`

3. [BUG] Template literal syntax error
   Location: src/components/ModalView.jsx:33
   Code: name={`${index === 1} ? 'small' : 'large`}
   Issue: Broken template literal - closing tick misplaced
   Fix: name={index === 1 ? 'small' : 'large'}

------------------------------------------------------------------------------
HIGH PRIORITY ISSUES
------------------------------------------------------------------------------

4. [BUG] Video import naming mismatch
   Location: src/utils/index.js:8-9
   Issue: Variable names don't match actual video files
   - highlightSecondVideo imports "hightlight-third.mp4"
   - highlightThirdVideo imports "hightlight-sec.mp4"
   Impact: Videos display in wrong order in carousel

5. [BUG] Typo in class name
   Location: src/components/Features.jsx:38
   Code: className="screen-max-wdith"
   Issue: Should be "screen-max-width"
   Impact: Layout styling not applied

6. [BUG] Typo in responsive class
   Location: src/components/Footer.jsx:6
   Code: className="py-5 ms:px-10 px-5"
   Issue: "ms:px-10" should be "sm:px-10"
   Impact: Responsive padding not working on small screens

7. [BUG] Invalid CSS class
   Location: src/components/Footer.jsx:11,14
   Code: className="underLine text-blue"
   Issue: "underLine" is not a valid Tailwind class
   Fix: Should be "underline"

8. [REACT] GSAP timeline created outside useEffect
   Location: src/components/Modal.jsx:33
   Code: const tl = gsap.timeline();
   Issue: Creates new timeline on every render
   Fix: Move inside useEffect or useMemo

------------------------------------------------------------------------------
MEDIUM PRIORITY ISSUES
------------------------------------------------------------------------------

9. [DEPENDENCY] Sentry package version mismatch
   Location: package.json
   Issue: @sentry/browser (v7.x) and @sentry/react (v8.x) mixed
   Fix: Upgrade @sentry/browser to v8.x or use consistent versions

10. [REACT] Missing useEffect cleanup
    Location: src/components/VideoCarousel.jsx:110-113
    Issue: gsap.ticker.add() called but cleanup incomplete
    Fix: Ensure proper cleanup in useEffect return

11. [REACT] Missing dependency in useEffect
    Location: src/components/VideoCarousel.jsx:116
    Dependencies: [videoId, startPlay]
    Issue: isPlaying is used but not in dependency array
    Fix: Add isPlaying to dependency array

12. [NAMING] File naming inconsistencies
    - Modal.jsx exports "Model" (component mismatch)
    - ModalView.jsx exports "ModelView" (component mismatch)
    Recommendation: Rename files to Model.jsx and ModelView.jsx

------------------------------------------------------------------------------
LOW PRIORITY / CODE QUALITY ISSUES
------------------------------------------------------------------------------

13. [REACT] Unnecessary React imports
    Locations:
    - src/components/Navbar.jsx:1
    - src/components/Features.jsx:2
    - src/components/HowItWorks.jsx:1
    - src/components/Footer.jsx:1

14. [REACT] Unused Sentry import
    Location: src/App.jsx:6
